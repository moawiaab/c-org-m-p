## About c-org-m-p

c = charitable
org = organization
m = management
p = project

## Description 
 Open source software for charitable organization management

## After Download
# 1
Run cp .env.example .env file to copy example file to .env
# 2
 ```
composer install
```
# 3
```
php artisan migrate --seed
```
# 4
```
php artisan key:g
```
# 5 

```
php artisan storage:link
```
# 6 run server
```
php artisan serve
```
